# OCTET_LENGTH

## Syntax

```sql
OCTET_LENGTH(str)
```

## Purpose

`OCTET_LENGTH()` is a synonym of `LENGTH()`. 

## Examples

```sql
obclient> SELECT OCTET_LENGTH('中国'), OCTET_LENGTH('hello');
+------------------------+-----------------------+
| OCTET_LENGTH('中国')   | OCTET_LENGTH('hello') |
+------------------------+-----------------------+
|                      6 |                     5 |
+------------------------+-----------------------+
1 row in set
```
