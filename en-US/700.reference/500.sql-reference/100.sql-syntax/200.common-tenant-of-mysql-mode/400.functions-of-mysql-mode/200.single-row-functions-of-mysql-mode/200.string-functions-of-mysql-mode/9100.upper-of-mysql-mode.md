# UPPER

## Syntax

```sql
UPPER(str)
```

## Purpose

`UPPER()` converts lowercase letters in the `str` string into uppercase letters.

## Examples

```sql
obclient> obclient> SELECT UPPER('obclienthello! ');
+----------------------------+
| UPPER('obclienthello! ')   |
+----------------------------+
| OBCLIENTHELLO!             |
+----------------------------+
1 row in set
```
