# COS

## Syntax

```sql
COS(X)
```

## Purpose

`COS()` returns the cosine of the `X` radian.

## Examples

```sql
obclient> SELECT COS(PI());
+-----------+
| COS(PI()) |
+-----------+
|        -1 |
+-----------+
1 row in set
```
