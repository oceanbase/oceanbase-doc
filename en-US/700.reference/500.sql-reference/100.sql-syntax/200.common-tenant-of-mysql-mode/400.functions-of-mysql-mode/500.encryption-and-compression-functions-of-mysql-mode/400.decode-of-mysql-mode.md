# DECODE

## Syntax

```sql
DECODE(crypt_str,pass_str)
```

## Purpose

`DECODE()` uses the `pass_str` password to decrypt the `crypt_str` string and returns the decrypted string. 

## Examples

```sql
obclient> SELECT DECODE(ENCODE('secret','cry'), 'cry');
+---------------------------------------+
| DECODE(ENCODE('secret','cry'), 'cry') |
+---------------------------------------+
| secret                                |
+---------------------------------------+
1 row in set
```
