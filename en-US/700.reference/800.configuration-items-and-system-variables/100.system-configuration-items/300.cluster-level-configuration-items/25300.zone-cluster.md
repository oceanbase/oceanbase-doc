zone
=========================

`zone` specifies the name of the zone where the node is located. This parameter is generally left unspecified.


| **Attribute** | **Description** |
|------------------|--------|
| Type | String |
| Default value | Empty string |
| Value range | NULL |
| Effective upon OBServer node restart | No |
