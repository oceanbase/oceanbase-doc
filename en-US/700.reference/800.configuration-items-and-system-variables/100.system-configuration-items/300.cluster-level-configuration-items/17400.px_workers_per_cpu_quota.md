# px_workers_per_cpu_quota


`px_workers_per_cpu_quota` specifies the proportion of parallel worker threads.


| **Attribute** | **Description** |
|------------------|-----------|
| Type | Integer |
| Default value | 10 |
| Value range | \[0, 20\] |
| Effective upon OBServer node restart | No |
