enable_sys_table_ddl
=========================================

`enable_sys_table_ddl` specifies whether to allow manual creation of system tables.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | <li> False   <li> True |
| Effective upon OBServer node restart | No |
