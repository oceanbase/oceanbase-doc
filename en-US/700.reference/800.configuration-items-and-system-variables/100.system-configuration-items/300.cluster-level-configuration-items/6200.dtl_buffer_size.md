dtl_buffer_size
====================================

`dtl_buffer_size` specifies the cache size allocated to the SQL data transfer module. You must specify the unit. By default, MB is used.


| **Attribute** | **Description** |
|------------------|-------------|
| Type | Capacity |
| Default value | 64K |
| Value range | \[4K, 2M\] |
| Effective upon OBServer node restart | No |
