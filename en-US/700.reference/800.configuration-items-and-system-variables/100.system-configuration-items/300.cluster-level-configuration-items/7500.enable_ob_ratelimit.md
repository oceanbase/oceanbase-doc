# enable_ob_ratelimit


`enable_ob_ratelimit` specifies whether to enable rate limiting between zones in RPC connections.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | String |
| Default value | False |
| Value range | <li> False   <li> True |
| Effective upon OBServer node restart | No |
