# enable_ddl


`enable_ddl` specifies whether to allow the execution of DDL statements.


| **Attribute** | Description |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
