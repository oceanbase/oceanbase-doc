# row_compaction_update_limit

`row_compaction_update_limit` specifies the number of data updates that triggers a row compaction in memory.


| **Attribute** | **Description** |
|------------------|-------------|
| Type | Integer |
| Default value | 6 |
| Value range | \[1, 6400\] |
| Effective upon OBServer node restart | No |
