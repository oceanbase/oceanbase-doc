# sys_bkgd_net_percentage


`sys_bkgd_net_percentage` specifies the percentage of bandwidth for background tasks.


| **Attribute** | **Description** |
|------------------|------------|
| Type | Integer |
| Default value | 60 |
| Value range | \[0, 100\] |
| Effective upon OBServer node restart | No |
