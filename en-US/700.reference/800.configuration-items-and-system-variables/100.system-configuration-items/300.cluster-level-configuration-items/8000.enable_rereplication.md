enable_rereplication
=========================================

`enable_rereplication` specifies whether to enable automatic re-replication.


| Attribute | Description |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
