enable_major_freeze
========================================

`enable_major_freeze` specifies whether to enable automatic global freezing.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
