enable_sql_audit
=====================================

`enable_sql_audit` specifies whether to enable SQL audit.


| **Attribute** | **Description** |
|------------------|---------------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> `True`: enable   <li> `False`: disable |
| Effective upon OBServer node restart | No |
