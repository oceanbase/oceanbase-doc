leak_mod_to_check
======================================

`leak_mod_to_check` is used to check memory leakage and is an internal debugging feature.


| **Attribute** | **Description** |
|------------------|--------|
| Type | String |
| Default value | none |
| Value range | NULL |
| Effective upon OBServer node restart | No |
