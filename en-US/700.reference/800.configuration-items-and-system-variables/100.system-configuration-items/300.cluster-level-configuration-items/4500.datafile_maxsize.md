# datafile_maxsize

`datafile_maxsize` specifies the maximum space allowed in automatic scaling for disk files. 

<main id="notice" type='explain'>

  <h4>Note</h4>

  <p>This parameter was introduced in OceanBase Database V3.1.5. </p>

</main>
