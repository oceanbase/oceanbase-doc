# enable_async_syslog


`enable_async_syslog` specifies whether to enable asynchronous writing of system logs.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
