enable_perf_event
======================================

`enable_perf_event` specifies whether to enable the information collection feature for performance events.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
