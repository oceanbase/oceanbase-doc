# arbitration_timeout

`arbitration_timeout` specifies the timeout period for triggering automatic downgrade. 

<main id="notice" type='explain'>

  <h4>Note</h4>

  <p>This parameter was introduced in OceanBase Database V4.1.0. </p>

</main>

| --- | --- |
