audit_sys_operations
=========================================

`audit_sys_operations` specifies whether to track the operations of system users.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | <li> False   <li> True |
| Effective upon OBServer node restart | No |
