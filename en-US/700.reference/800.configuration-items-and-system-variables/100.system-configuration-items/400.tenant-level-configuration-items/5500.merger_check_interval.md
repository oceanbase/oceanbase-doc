# merger_check_interval

`merger_check_interval` specifies the interval for scheduling the thread for checking the major compaction status.


| **Attribute** | **Description** |
|------------------|--------------|
| Type | Time |
| Default value | 10m |
| Value range | \[10s, 60m\] |
| Effective upon OBServer node restart | No |
