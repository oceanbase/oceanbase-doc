# enable_monotonic_weak_read

`enable_monotonic_weak_read` specifies whether to enable monotonic reads.


| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | <li> True   <li> False |
| Effective upon OBServer node restart | No |
