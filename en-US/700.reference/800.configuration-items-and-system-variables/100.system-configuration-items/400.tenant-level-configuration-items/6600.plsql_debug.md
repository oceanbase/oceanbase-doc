plsql_debug
================================

`plsql_debug` specifies whether to compile code for debugging.


| Attribute | Description |
|------------------|------------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | <li> `True`   <li> `False` |
| Effective upon OBServer node restart | No |



Here is an example:

```sql
obclient> ALTER SYSDATE SET plsql_debug=false;
```
