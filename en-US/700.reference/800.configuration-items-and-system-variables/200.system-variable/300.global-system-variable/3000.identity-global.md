# identity

`identity` is synonymous with the `last_insert_id` variable and can be queried by using `select @@identity`.

| **Attribute** | **Description** |
|--------|-----------------------------|
| Type | UINT |
| Default value | 0 |
| Value range | \[0, 18446744073709551615\] |
| Effective scope | Session |
