# connect_timeout

`connect_timeout` specifies the connection timeout period, in seconds.

| **Attribute** | **Description** |
|--------|------------------------------------------------------------------------------------------------------------|
| Type | Integer |
| Default value | 10 |
| Value range | \[2, 31536000\] |
| Effective scope | Global |
