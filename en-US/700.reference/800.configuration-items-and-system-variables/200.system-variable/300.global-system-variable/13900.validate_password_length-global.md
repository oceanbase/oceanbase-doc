# validate_password_length

`validate_password_length` specifies the minimum length of a user password.

| **Attribute** | **Description** |
|---------|------------------|
| Type | UINT |
| Default value | 0 |
| Value range | \[0, 2147483647\] |
| Effective scope | Global |
| Involved in serialization | Yes |
