# nls_sort

`nls_sort` specifies the string value sorting rule.

<main id="notice" type='explain'>
    <h4>Note</h4>
    <p>This variable cannot be modified. </p>
</main>

|----------|------------------------------------------------------------------------------------------------------------|
| Type | Varchar |
| Default value | BINARY |
| Value range | N/A |
| Applicable scope | <li> Global   <li> Session |
