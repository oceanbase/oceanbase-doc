# default_password_lifetime

`default_password_lifetime` specifies the permitted password lifetime.

| **Attribute** | **Description** |
|---------|-------------|
| Type | UINT |
| Default value | 0 |
| Value range | \[0, 65535\] |
| Effective scope | Global |
| Involved in serialization | Yes |
