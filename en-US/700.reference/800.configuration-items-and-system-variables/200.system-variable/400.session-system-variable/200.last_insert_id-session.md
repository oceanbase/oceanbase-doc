# last_insert_id

`last_insert_id` returns the last inserted auto-increment field value in the current session.

| **Attribute** | **Description** |
|--------|-----------------------------|
| Type | UINT |
| Default value | 0 |
| Value range | \[0, 18446744073709551615\] |
| Effective scope | Session |
