# timestamp

`timestamp` specifies the timestamp in seconds. This variable affects the result of `SELECT NOW(6)`.

| **Attribute** | **Description** |
|--------|--------------------|
| Type | Numeric |
| Default value | 0 |
| Value range | \[0, 253402272000) |
| Effective scope | Session |
