# oceanbase.CDB_TABLES

## Purpose

The `oceanbase.CDB_TABLES` view displays the information about tables in all tenants.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
