# oceanbase.DBA_PART_TABLES

## Purpose

The `oceanbase.DBA_PART_TABLES` view displays information about partitioned tables.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
