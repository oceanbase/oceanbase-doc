# oceanbase.CDB_INDEXES

## Purpose

The `oceanbase.CDB_INDEXES` view displays the indexes of all tenants.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
