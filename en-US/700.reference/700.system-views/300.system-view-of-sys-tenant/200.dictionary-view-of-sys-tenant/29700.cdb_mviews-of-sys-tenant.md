# CDB_MVIEWS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V4.3.0. </p>
</main>

## Purpose

The `CDB_MVIEWS` view displays information about all materialized views.

## Columns
