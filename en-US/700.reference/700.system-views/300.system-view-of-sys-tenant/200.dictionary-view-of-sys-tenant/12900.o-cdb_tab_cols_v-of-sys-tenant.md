# oceanbase.CDB_TAB_COLS_V$

## Purpose

The `oceanbase.CDB_TAB_COLS_V$` view displays the column information of the table and view objects in all tenants.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
