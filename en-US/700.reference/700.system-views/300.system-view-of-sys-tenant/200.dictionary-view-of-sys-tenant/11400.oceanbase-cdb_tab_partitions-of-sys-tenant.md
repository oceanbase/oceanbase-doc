# oceanbase.CDB_TAB_PARTITIONS

## Purpose

The `oceanbase.CDB_TAB_PARTITIONS` view provides the partitioning information of the partitioned tables of all tenants.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
