USER_TABLES
================================

Purpose
-----------

The `USER_TABLES` view displays all tables owned by the current user.

<main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only MySQL mode. </p>
  </main>

Related views

* DBA_TABLES
