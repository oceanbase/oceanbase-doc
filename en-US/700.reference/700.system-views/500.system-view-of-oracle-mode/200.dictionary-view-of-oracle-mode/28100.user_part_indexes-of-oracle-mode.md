USER_PART_INDEXES
======================================

Purpose
-----------

The `USER_PART_INDEXES` view displays the partitioning information about partitioned indexes owned by the current user.

<main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only MySQL mode. </p>
  </main>

Related views
-------------

* DBA_PART_INDEXES

* USER_PART_INDEXES
