max_px_worker_count 
========================================

max_px_worker_count specifies the maximum number of threads for the SQL parallel query engine. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 64              |
| Value range                     | \[0, 65535\]    |
| Effective upon OBServer restart | No              |
