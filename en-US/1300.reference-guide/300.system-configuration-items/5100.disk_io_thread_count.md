disk_io_thread_count 
=========================================

disk_io_thread_count specifies the number of disk I/O threads. The value must be an even number. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 8               |
| Value range                     | \[2, 32\]       |
| Effective upon OBServer restart | No              |
