# 内存相关内部表

OceanBase 数据库支持通过内部表来查看内存的使用情况。

|                  视图                  | 描述 |   |
|--------------------------------------|----|---|
| __all_virtual_memory_info            | OBServer 内存标签的统计信息。 ||
| __all_virtual_tenant_memstore_info   | MemStore 统计信息。 ||
| __all_virtual_kvcache_info           | kvcache 统计信息。 ||
