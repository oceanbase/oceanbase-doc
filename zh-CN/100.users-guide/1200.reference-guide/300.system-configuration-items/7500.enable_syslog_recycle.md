# enable_syslog_recycle

`enable_syslog_recycle` 用于是否打开记录启动前的旧日志的开关。与 `max_syslog_file_count` 配合使用，决定回收逻辑是否考虑旧的日志文件。

|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | <ul><li>True</li><li>False</li></ul>    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>仅当 <code>max_syslog_file_count</code> 配置项的值设置为非 0 正数时，该功能才会生效。</p>
</main>
