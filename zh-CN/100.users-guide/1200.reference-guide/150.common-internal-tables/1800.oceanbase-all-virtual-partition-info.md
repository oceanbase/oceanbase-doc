# oceanbase.__all_virtual_partition_info

## 功能

用于展示了 Partition Group 在事务、clog 等层面的基本属性。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p> 该内部表仅支持在 sys 租户下查询。</p>
</main>

| svr_port |  bigint(20) | NO | 服务器端口号 |
