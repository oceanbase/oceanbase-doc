transaction_isolation 
==========================================

`transaction_isolation` 用于设置事务的隔离级别。


| **属性**  |                                                                                                               **描述**                                                                                                               |
|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                                                                                                                                            |
| 默认值     | READ-COMMITTED                                                                                                                                                                                                                     |
| 取值范围    | <ul><li>READ-UNCOMMITTED</li><li>READ-COMMITTED</li><li>REPEATABLE-READ</li><li>SERIALIZABLE</li></ul>    |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>                                                                                                                         |
| 是否参与序列化 | 是                                                                                                                                                                                                                                  |
