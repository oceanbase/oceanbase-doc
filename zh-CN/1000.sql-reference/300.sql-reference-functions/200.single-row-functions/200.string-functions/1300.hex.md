HEX 
========================



声明 
-----------------------

```javascript
HEX(str)
```



说明 
-----------------------

将数字或字符串 `str` 转化为十六进制字符串，其中 `str` 中的每个字符的每个字节都转换为两个十六进制数字。

对于数字类型的参数 `N`，`UNHEX(N)` 并不是 `HEX(N)` 的逆运算，对于这种情况，您可以使用 `CONV(HEX(N),16,10)` 替代。

示例 
-----------------------

```javascript
obclient> SELECT HEX(255), HEX('abc');
+----------+------------+
| HEX(255) | HEX('abc') |
+----------+------------+
| FF       | 616263     |
+----------+------------+
1 row in set (0.00 sec)
```
