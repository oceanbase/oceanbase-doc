LCASE 
==========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/LCASE


声明 
-----------------------

```sql
LCASE(str)
```



说明 
-----------------------

将字符串中的大写字母转化为小写字母。

该函数是 `LOWER()` 的同义词。

示例 
-----------------------

```sql
obclient> SELECT LCASE('OBCLIENT您好！');
+----------------------------+
| LCASE('OBCLIENT您好！')    |
+----------------------------+
| obclient您好！             |
+----------------------------+
1 row in set 
```
