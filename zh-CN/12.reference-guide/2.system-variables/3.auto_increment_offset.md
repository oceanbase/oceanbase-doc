auto_increment_offset 
==========================================

auto_increment_offset 用于确定 AUTO_INCREMENT 列值的起点。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | uint                                                                                                       |
| 默认值    | 1                                                                                                          |
| 取值范围   | \[1, 65535\]                                                                                               |
| 生效范围   | <ul><li>Global</li><li>Session</li></ul>    |
