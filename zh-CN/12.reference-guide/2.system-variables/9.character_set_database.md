character_set_database 
===========================================

character_set_database 用于设置默认数据库的字符集。


| **属性** |                                                                                                                   **描述**                                                                                                                    |
|--------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                                                                                                                                                                                      |
| 默认值    | utf8mb4                                                                                                                                                                                                                                     |
| 取值范围   | <ul><li> utf8mb4  </li><li> binary  </li><li> gbk  </li><li> gb18030  </li><li>utf16</li></ul>     |
| 生效范围   | <ul><li>Global</li><li>Session</li></ul>                                                                                                                                  |
| 是否可以为空 | 是                                                                                                                                                                                                                                           |
