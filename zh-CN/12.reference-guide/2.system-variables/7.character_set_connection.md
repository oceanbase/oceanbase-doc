character_set_connection 
=============================================

character_set_connection 用于设置收到语句后应转换的字符集。


| **属性**  |                                                                       **描述**                                                                        |
|---------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | string                                                                                                                                              |
| 默认值     | utf8mb4                                                                                                                                             |
| 取值范围    | <ul><li> utf8mb4  </li><li> binary  </li><li> gbk </li></ul>   |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>                                            |
| 是否参与序列化 | 是                                                                                                                                                   |
| 是否可以为空  | 是                                                                                                                                                   |
