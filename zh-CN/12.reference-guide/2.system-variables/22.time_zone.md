time_zone 
==============================

time_zone 用于设置当前租户会话使用的时区，支持 `'+08:00' `的 offset 形式和 `Asia/Shanghai `的地域形式取值。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | +8:00                                                                                                      |
| 取值范围    | -12:59 \~ +13:00                                                                                           |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>    |
| 是否参与序列化 | 是                                                                                                          |
