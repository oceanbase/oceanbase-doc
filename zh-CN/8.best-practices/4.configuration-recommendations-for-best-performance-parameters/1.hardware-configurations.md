硬件配置说明 
===========================



作为原生的分布式关系数据库，OceanBase 数据库具有极强的可扩展性，服务器的硬件配置越高，数据库服务的性能也会越好。

本文以下表中的服务器配置为例，提供最佳性能配置推荐指导。


| 服务器配置项 |          描述          |
|--------|----------------------|
| 服务器数量  | 3 台                  |
| 芯片     | Intel                |
| CPU    | 32 Core              |
| 内存     | 128 GB               |
| 磁盘     | 2 TB SSD 存储          |
| 网卡     | 万兆网卡                 |
| 操作系统   | Linux kernel 3.1 及以上 |
| 文件系统   | 推荐 EXT4 戓 XFS        |
