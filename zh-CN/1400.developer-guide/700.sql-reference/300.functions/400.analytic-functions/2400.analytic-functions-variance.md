# VARIANCE

## 声明

```sql
VARIANCE([ ALL ] expr) OVER (analytic_clause)
```

## 说明

返回表达式 `expr` 的总体标准方差，该函数是 **VAR_POP** 的别名。

如果没有匹配到行，则该函数返回 `NULL`。

## 示例
