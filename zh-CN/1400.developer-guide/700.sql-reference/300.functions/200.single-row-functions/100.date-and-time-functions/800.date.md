# DATE

## 声明

```sql
DATE(expr)
```

## 说明

返回时间或时间日期表达式 `expr` 的日期。

## 示例

```sql
obclient> SELECT DATE('2021-07-14 19:12:03');
+-----------------------------+
| DATE('2021-07-14 19:12:03') |
+-----------------------------+
| 2021-07-14                  |
+-----------------------------+
1 row in set
```
