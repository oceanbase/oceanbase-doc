# nls_numeric_characters

`nls_numeric_characters` 用于控制 number 和字符串转换中字符串中小数分隔符和千分位分隔符的字符。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量不支持修改。</p>
  </main>

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                 |
| 默认值     | 空字符串                    |
| 取值范围    | N/A                     |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                       |
