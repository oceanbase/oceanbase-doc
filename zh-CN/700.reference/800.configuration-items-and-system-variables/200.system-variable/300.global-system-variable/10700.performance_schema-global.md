# performance_schema

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量仅用于兼容 MySQL 8.0，相关功能并不支持，可以查询及设置，设置行为不报错但不生效。</p>
</main>

## 功能描述

`performance_schema` 用于向客户端声明是否支持 performance 信息的查询。

## 属性说明

|   属性       |      描述                                             |
|--------------|-------------------------------------------------------|
| 参数类型     | Bool                   |
| 默认值       | OFF                    |
| 取值范围     | <ul><li>ON：支持</li><li>OFF：不支持</li></ul>    |
| 生效范围     | GLOBAL                 |
| 是否可修改   | 是，支持通过 `SET` 语句修改。|
