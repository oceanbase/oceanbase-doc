# event_scheduler

`event_scheduler` 用于确定事件调度程序是否在服务器上启用和运行。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V3.2.4 版本新增，仅在 MySQL 模式下用于兼容。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | bool |
| 默认值 | OFF |
| 取值范围 | <ul><li> OFF：事件调度程序已停止。 </li><li> ON：事件调度程序已启动；事件调度程序线程运行并执行所有计划的事件。 </li></ul>|
| 生效范围 |GLOBAL |
|是否可修改|否|
