# optimizer_features_enable

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V4.2.0 版本开始引入。</p>
</main>

## 功能描述

`optimizer_features_enable` 用于启用不同 OceanBase 版本优化器功能。

## 属性说明

| 属性 | 描述 |
| --- | --- |
| 参数类型 | varchar |
| 默认值 | 空 |
| 取值范围 | '4.0.0.0' 及后续版本号 |
| 生效范围 | <ul><li>Global  </li><li>Session </li></ul>|
