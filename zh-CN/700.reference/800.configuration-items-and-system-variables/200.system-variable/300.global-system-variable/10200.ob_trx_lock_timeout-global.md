# ob_trx_lock_timeout

ob_trx_lock_timeout 用于设置事务的等锁超时时长，单位为微秒。

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                     |
| 默认值     | -1  表示不生效                    |
| 取值范围    | 有效取值范围：\[0,+∞) <main id="notice" type='explain'><h4>说明</h4><p>0 表示不设置等待时长</p></main>         |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                       |
