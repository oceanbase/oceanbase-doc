# sql_safe_updates

sql_safe_updates 用于设置是否启用 MySQL 的 SQL 安全模式，默认为关闭。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量仅用于兼容，OceanBase 数据库当前暂未实现该变量的功能。</p>
  </main>

| **属性**  |                                                          **描述**                                                          |
|---------|--------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                  |
| 默认值     | 0                                     |
| 取值范围    | <li> 0：表示 FALSE，即关闭   <li> 1：表示 TRUE，即开启    |
| 生效范围    | <li> GLOBAL   <li> SESSION                  |
| 是否参与序列化 | 是                                     |
