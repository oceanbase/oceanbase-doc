# datadir

`datadir` 用于查看数据存储的本地磁盘路径。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量仅用于兼容，OceanBase 数据库当前暂未实现该变量的功能。</p>
  </main>

| **属性** |         **描述**         |
|--------|------------------------|
| 参数类型   | varchar                |
| 默认值    | /usr/local/mysql/data/ |
| 取值范围   | N/A                    |
| 生效范围   | Global                 |
| 是否可修改  | 否                      |
