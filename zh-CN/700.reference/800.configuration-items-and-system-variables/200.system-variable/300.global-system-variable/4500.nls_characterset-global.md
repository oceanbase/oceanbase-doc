# nls_characterset

`nls_characterset` 用于查看数据库中 CHAR、VARCHAR2、CLOB 等数据类型的默认字符集。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量当前暂不生效。</p>
  </main>

|  **属性**  |                                                                             **描述**                                                                              |
|----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型     | varchar                                                                      |
| 默认值      | AL32UTF8                                                                     |
| 取值范围     | N/A   |
| 生效范围     | Global                                                                       |
| 是否参与序列化  | 是                                                                            |
| 是否影响计划生成 | 是                                                                            |
| 是否可修改    | 否  |
