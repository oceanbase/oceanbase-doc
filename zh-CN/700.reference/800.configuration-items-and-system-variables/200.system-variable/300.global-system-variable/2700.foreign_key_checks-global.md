# foreign_key_checks

`foreign_key_checks` 用于设置是否在 DML/DDL 中检查外建约束。

| **属性**  |                                                     **描述**                                                     |
|---------|----------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                        |
| 默认值     | 1                           |
| 取值范围    | <li> 0：不检查外键约束   <li> 1：检查外键约束    |
| 生效范围    | <li> Global   <li> Session        |
| 是否参与序列化 | 是                           |
