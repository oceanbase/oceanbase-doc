# ob_plan_cache_evict_low_percentage

ob_plan_cache_evict_low_percentage 用于设置停止淘汰计划缓存的内存大小占内存上限绝对值的百分比。

| **属性**  |   **描述**   |
|---------|------------|
| 参数类型    | int        |
| 默认值     | 50         |
| 取值范围    | \[0, 100\] |
| 生效范围    | GLOBAL     |
| 是否参与序列化 | 是          |

停止淘汰计划缓存的内存大小（淘汰计划的低水位线）= 内存上限绝对值 \* ob_plan_cache_evict_low_percentage/100。有关内存上限绝对值的相关介绍，请参见 [ob_plan_cache_percentage](../300.global-system-variable/8600.ob_plan_cache_percentage-global.md)。
