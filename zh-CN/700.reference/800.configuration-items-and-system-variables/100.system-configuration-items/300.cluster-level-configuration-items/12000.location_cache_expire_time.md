# location_cache_expire_time

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本进行了架构的变更升级，该配置项已不再适用，从 V4.0.0 版本开始删除。</p>
</main>

`location_cache_expire_time` 用于设置分区位置缓存中分区位置信息的过期时长。不建议修改。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | Time      |
| 默认值              | 600s      |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
