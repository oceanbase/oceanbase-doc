minor_merge_concurrency 
============================================

`minor_merge_concurrency` 用于设置转储时的并发线程数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 0，表示并发线程数为 min[10，cpu_cnt*0.3]        |
| 取值范围             | [0, 64] |
| 是否重启 OBServer 节点生效 | 否         |
