# clog_cache_priority

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对架构进行了变更升级，事务日志不再使用 Cache，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`clog_cache_priority` 用于设置事务日志占用缓存的优先级。

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[1, +∞) |
| 是否重启 OBServer 节点生效 | 否        |
