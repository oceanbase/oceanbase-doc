# ob_esi_memory_limit

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对物理备份恢复功能进行了架构的优化调整，该配置项对应的功能在 V4.x 版本中不再支持，从 V4.0.0 版本开始删除。</p>
</main>

`ob_esi_memory_limit` 用于设置 obesi 进程的内存使用上限。

| **属性** | **描述** |
| --- | --- |
| 参数类型 | Capacity |
| 默认值 | 0M |
| 取值范围 | [64M, +∞) |
| 是否重启 OBServer 节点生效 | 否 |
