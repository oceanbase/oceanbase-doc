# zone_merge_order

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本不再支持按 Zone 控制合并，也不再支持轮转合并，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`zone_merge_order` 用于设置 Zone 的轮转合并顺序。不指定时，由系统自动决定。

|      **属性**      | **描述** |
|------------------|---------------------|
| 参数类型                      | String  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
