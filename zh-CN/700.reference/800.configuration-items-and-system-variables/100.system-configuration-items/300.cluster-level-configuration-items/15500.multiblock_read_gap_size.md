# multiblock_read_gap_size

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本进行了存储架构的变更，该配置项已不再适用，从 V4.0.0 版本开始删除。 </p>
</main>

`multiblock_read_gap_size` 用于设置一次 IO 聚合读取块缓存的最大容量。

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型          | Capacity       |
| 默认值              | 0K         |
| 取值范围             | \[0k, 2M\] |
| 是否重启 OBServer 节点生效 | 否          |
