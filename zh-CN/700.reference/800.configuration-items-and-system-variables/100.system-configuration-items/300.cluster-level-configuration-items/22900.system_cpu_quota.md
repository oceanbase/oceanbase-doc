# system_cpu_quota

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本进行了架构的变更升级，该配置项已不再适用，从 V4.0.0 版本开始删除。</p>
</main>

`system_cpu_quota` 用于设置系统租户可使用的 CPU 配额。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值              | 10        |
| 取值范围             | \[0, 16\] |
| 是否重启 OBServer 节点生效 | 否         |
