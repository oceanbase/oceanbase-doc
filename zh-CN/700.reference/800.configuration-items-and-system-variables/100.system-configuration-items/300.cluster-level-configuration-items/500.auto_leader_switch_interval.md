# auto_leader_switch_interval

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对系统架构进行了优化调整，选举不再需要设置刷新间隔，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`auto_leader_switch_interval` 用于设置自动切主后台线程工作间隔时间。

|     **属性**     |  **描述**  |
|----------------|----------|
| 参数类型           |  Time     |
| 默认值            | 30s，表示 30 秒       |
| 取值范围           | \[1s，+∞) |
| 是否需要重启OBserver | 否        |
