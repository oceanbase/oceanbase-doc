# merger_completion_percentage

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于该配置项自引入后使用率较低，经过综合评估，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`merger_completion_percentage` 用于设置合并完成副本数的百分比阈值，达到该阈值则认为本轮合并完成调度。

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | Int         |
| 默认值              | 100        |
| 取值范围             | \[5, 100\] |
| 是否重启 OBServer 节点生效 | 否          |
