# trx_force_kill_threshold

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>由于在 V4.x 版本中对应逻辑已改变，对应功能不再使用，该配置项从 V4.0.0 版本开始删除。</p>
</main>

## 功能描述

`trx_force_kill_threshold` 用于设置因冻结或切主需要杀事务时的最长等待时间。

## 属性说明

|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 100ms        |
| 取值范围             | \[1ms, 10s\] |
| 是否重启 OBServer 节点生效 | 否            |
