# enable_election_group

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对系统架构进行了优化调整，不再有选举组的概念，该配置项从 V4.0.0 版本开始弃用。</p>
</main>
`enable_election_group` 用于设置是否打开选举组策略。

|        属性        |                                                   描述                                                   |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | Bool                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | </li><li> True   </li><li> False    |
| 是否重启 OBServer 节点生效 | 否                                                                                                      |
