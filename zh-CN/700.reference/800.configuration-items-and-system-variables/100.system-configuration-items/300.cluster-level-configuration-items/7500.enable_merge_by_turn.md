# enable_merge_by_turn

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本中不再支持轮转合并功能，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`enable_merge_by_turn` 用于设置是否开启轮转合并策略。

|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | Bool                                                                                                  |
| 默认值              | False                                                                                                  |
| 取值范围             | </li><li> True   </li><li> False    |
| 是否重启 OBServer 节点生效 | 否                                                                                                      |
