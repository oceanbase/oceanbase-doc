# zone_merge_timeout

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本不再支持按 Zone 控制合并，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`zone_merge_timeout` 用于设置单个 zone 的合并超时时间。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型                      | Time      |
| 默认值              | 3h        |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
