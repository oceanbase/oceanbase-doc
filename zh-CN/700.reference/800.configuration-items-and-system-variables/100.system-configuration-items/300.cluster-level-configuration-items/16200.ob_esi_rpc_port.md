# ob_esi_rpc_port

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对物理备份恢复功能进行了架构的优化调整，该配置项对应的功能在 V4.x 版本中不再支持，从 V4.0.0 版本开始删除。</p>
</main>

`ob_esi_rpc_port` 用于设置 obesi 进程和 observer 进程之间的通信端口。

| **属性**    | **描述**        |
|-------------|-----------------|
| 参数类型 | Int |
| 默认值 | 2501 |
| 取值范围 | [1025, 65535] |
| 是否重启 OBServer 节点生效 | 否 |
