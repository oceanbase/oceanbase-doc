# tenant_groups

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本中不再支持租户组功能，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`tenant_groups` 用于设置负载均衡策略中使用的租户组。

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型                      | String  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
