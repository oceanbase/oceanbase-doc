# partition_table_check_interval

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本中不再有分区副本的概念，该配置项已不再适用，从 V4.0.0 版本开始删除。</p>
</main>

`partition_table_check_interval` 用于设置 OBServer 从分区表中删除不存在的副本的时间间隔。

|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | Time         |
| 默认值              | 30m，表示 30 分钟 |
| 取值范围             | \[1m, +∞)    |
| 是否重启 OBServer 节点生效 | 否            |
