# location_cache_priority

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本进行了架构的变更升级，该配置项已不再适用，从 V4.0.0 版本开始删除。</p>
</main>

`location_cache_priority` 用于设置位置缓存在系统缓存服务中的优先级。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | Int        |
| 默认值              | 1000      |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
