# enable_smooth_leader_switch

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本不再支持分区 Leader 切换，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`enable_smooth_leader_switch` 是平滑切主特性的开关。

|      **属性**      |                  **描述**                                 |
|------------------|------------------------------------------------------------|
| 参数类型             | Bool                                                |
| 默认值              | True                                                  |
| 取值范围             | </li><li> True   </li><li> False    |
| 是否重启 OBServer 节点生效 | 否                                               |
