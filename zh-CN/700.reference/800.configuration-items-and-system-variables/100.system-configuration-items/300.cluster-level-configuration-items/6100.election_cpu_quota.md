# election_cpu_quota

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对系统架构进行了优化调整，选举定时任务的线程数量随日志流数量的变化而变化，仅需要一个线程即可，不需要再配置 CPU 配额，该配置项从 V4.0.0 版本开始删除。</p>
</main>

`election_cpu_quota` 用于设置给副本选举相关的后台工作分配的 CPU 配额。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型                     | Double    |
| 默认值              | 3         |
| 取值范围             | \[0, 10\] |
| 是否重启 OBServer 节点生效 | 否         |
