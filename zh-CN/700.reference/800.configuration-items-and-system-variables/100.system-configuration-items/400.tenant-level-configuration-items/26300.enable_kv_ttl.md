# enable_kv_ttl

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V3.1.4 版本开始引入。</p>
</main>

## 功能描述

`enable_kv_ttl` 用于控制是否启动 TTL 后台任务，该配置项对周期性 TTL 任务有效，用户管控命令不受此配置项限制。

| 属性 | 描述 |
| --- | --- |
| 参数类型 | 布尔类型 |
| 默认值 | False |
| 取值范围 | <ul><li> True : 启动</li><li> False :  不启动 </li></ul>|
| 是否重启 OBServer 生效 | 否 |

## 配置示例

```sql
alter system set enable_kv_ttl= false;
```
