# freeze_trigger_percentage 

`freeze_trigger_percentage` 用于设置触发全局冻结的租户使用内存阈值。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始默认值由 70 调整为 20。</p>
</main>


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 20        |
| 取值范围             | (0, 100) |
| 是否重启 OBServer 节点生效 | 否         |
