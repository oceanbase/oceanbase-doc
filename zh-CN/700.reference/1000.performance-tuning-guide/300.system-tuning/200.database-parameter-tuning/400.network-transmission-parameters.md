|                     配置项                      |               描述                |                        建议                        |
|----------------------------------------------|---------------------------------|--------------------------------------------------|
| __easy_memory_limit                          | 发往单个 observer 的 rpc packet 内存上限 | 默认 4 G，大查询建议调大                                   |
| ob_proxy_readonly_transaction_routing_policy | 控制 Proxy 对于事务的路由是否受只读语句的影响      | 建议将取值修改为 False，表示 Proxy 对于事务的路由以第一条实际开启事务的语句为准   |
|
