## 配置模板

KV 的相关配置文件模板如下：

* basic_auth.yaml，基础认证相关的 KV 配置项。
* common_meta.yaml，元信息的 KV 配置项。
* log.yaml，agent 日志相关的 KV 配置项。
* ob_logcleaner.yaml，observer 日志清理相关的 KV 配置项。
* monagent_pipeline.yaml，流水线相关的 KV 配置项。
