# GV$ACTIVE_SESSION_HISTORY

<main id="notice" type='explain'>
  <h4>说明</h4>
  <ul><li>该视图从 V4.2.2 版本开始更名为<a href="800.gv-ob_active_session_history-of-mysql-mode.md">GV$OB_ACTIVE_SESSION_HISTORY</a>。</li>
  <li>该视图从 V4.0.0 版本开始引入。</li></ul>
</main>

## 功能

展示活跃会话历史记录。
