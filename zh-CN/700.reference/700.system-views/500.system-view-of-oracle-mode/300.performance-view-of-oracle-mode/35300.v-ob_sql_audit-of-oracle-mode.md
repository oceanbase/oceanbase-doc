## 功能描述

视图 `V$OB_SQL_AUDIT` 用于展示当前 OBServer 节点的每一次 SQL 请求的来源、执行状态等统计信息。该视图是按照租户拆分的，除了系统租户，其他租户不能跨租户查询。
