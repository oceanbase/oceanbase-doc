# GV$SQL_PLAN_MONITOR

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V1.4 版本开始引入。</p>
</main>

## 功能描述

视图 `GV$SQL_PLAN_MONITOR` 用于展示本租户下所有 OBServer 节点慢查询的 Operation 层面的统计，每个慢查询的每个 Operation 都会提供统计信息，展示算子消耗的时间、吐行等，同时也有 TRACE_ID 等标识信息。
