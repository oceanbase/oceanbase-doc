# DBMS_LOCK_ALLOCATED

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V4.2.1 版本开始引入。</p>
</main>

## 功能描述

展示已申请且未过期的 `DBMS_LOCK` 信息。

## 字段说明

| **字段** | **类型** | **是否为 NULL** | **描述** |
| --- | --- | --- | --- |
| NAME       | VARCHAR2(128)                     | NO   | 锁名称，Primary Key |
| LOCKID     | NUMBER(38)                        | NO   | 锁 ID |
| EXPIRATION | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO   | 锁超时时间 |
