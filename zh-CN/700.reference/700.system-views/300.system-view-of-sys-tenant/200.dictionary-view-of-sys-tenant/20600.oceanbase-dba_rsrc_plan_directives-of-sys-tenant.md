# oceanbase.DBA_RSRC_PLAN_DIRECTIVES

## 功能

展示数据库中每组资源计划指令的详细信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V3.1.2 版本开始引入。</p>
</main>

## 相关视图

* `oceanbase.DBA_RSRC_PLANS`

* `V$RSRC_PLAN`
