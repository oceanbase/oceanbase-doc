CURRENT_DATE
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/CURRENT_DATE-1


声明
-----------------------

```sql
CURRENT_DATE()
CURRENT_DATE
```



说明
-----------------------

返回当前日期，不含时间部分。该函数是 [CURDATE](../100.date-and-time-functions-1/300.CURDATE.md) 的别名。

示例
-----------------------

```javascript
obclient> SELECT CURRENT_DATE;
+--------------+
| CURRENT_DATE |
+--------------+
| 2021-08-23   |
+--------------+
1 row in set

obclient> SELECT CURRENT_DATE();
+----------------+
| CURRENT_DATE() |
+----------------+
| 2021-08-23     |
+----------------+
1 row in set
```
