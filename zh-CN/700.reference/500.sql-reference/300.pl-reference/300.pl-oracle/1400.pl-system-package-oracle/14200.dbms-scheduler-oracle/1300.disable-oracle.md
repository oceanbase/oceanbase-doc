# DISABLE

`DISABLE` 用于禁用定时分区均衡。

## 语法

```sql
DBMS_SCHEDULER.DISABLE('SCHEDULED_TRIGGER_PARTITION_BALANCE');
```

## 示例

```shell
obclient> CALL DBMS_SCHEDULER.DISABLE('SCHEDULED_TRIGGER_PARTITION_BALANCE');
Query OK, 0 rows affected (0.010 sec)
```
