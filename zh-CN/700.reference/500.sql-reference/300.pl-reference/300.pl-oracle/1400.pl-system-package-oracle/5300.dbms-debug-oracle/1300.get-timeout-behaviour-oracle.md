# GET_TIMEOUT_BEHAVIOUR

`GET_TIMEOUT_BEHAVIOUR` 函数用于返回当前超时行为。此调用是在目标会话中进行的。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法

```sql
DBMS_DEBUG.GET_TIMEOUT_BEHAVIOUR
RETURN BINARY_INTEGER;
```
