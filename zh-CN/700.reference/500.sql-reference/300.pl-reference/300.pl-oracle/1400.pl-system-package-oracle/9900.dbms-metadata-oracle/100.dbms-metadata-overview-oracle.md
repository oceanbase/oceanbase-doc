# DBMS_METADATA 概述

`DBMS_METADATA` 系统包用于创建 DDL，或通过提交 XML 文档以执行 DDL。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## DBMS_METADATA 权限说明

该系统包属于 `SYS` 用户，并具有授予 `PUBLIC` 的执行访问权限。`DBMS_METADATA` 通过调用者权限（即调用者的安全配置文件）运行。
