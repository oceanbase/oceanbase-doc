# PORT_STRING

`PORT_STRING` 函数用于返回一个字符串，用于标识操作系统和数据库的 `TWO TASK PROTOCOL` 版本。

字符串的最大长度根据端口设置。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法

```sql
DBMS_UTILITY.PORT_STRING 
RETURN VARCHAR2;
```
