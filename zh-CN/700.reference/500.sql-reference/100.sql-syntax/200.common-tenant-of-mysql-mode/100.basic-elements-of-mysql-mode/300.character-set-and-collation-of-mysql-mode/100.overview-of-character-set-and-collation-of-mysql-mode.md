# 概述

字符集是一组符号和编码。字符序是一组用于比较字符集中字符的规则。

关于字符集和字符序，OceanBase 数据库支持如下功能：

* 使用各种字符集存储字符串。

* 使用各种排序规则比较字符串。

* 在同一个服务器、同一个数据库甚至同一个表中混合使用不同字符集或字符序的字符串。

* 在任何级别启用字符集和字符序的规范。
