# SM3

## 声明

```javascript
SM3(bitstring)
```

## 说明

SM3 是中国国家密码管理局发布的一种哈希值计算方法，输入可以是任意长度的比特串，返回值为 256 比特的哈希值。

## 示例

```javascript
obclient> SELECT SM3(1);
+------------------------------------------------------------------+
| sm3(1)                                                           |
+------------------------------------------------------------------+
| cbdddb8e8421b23498480570d7d75330538a6882f5dfdc3b64115c647f3328c4 |
+------------------------------------------------------------------+
1 row in set
```
