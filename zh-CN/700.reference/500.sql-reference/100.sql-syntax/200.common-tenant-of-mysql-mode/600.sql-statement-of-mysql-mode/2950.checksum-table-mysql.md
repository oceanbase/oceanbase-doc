# CHECKSUM TABLE

该语句用于获取一张表的校验和（checksum）。通过计算表的内容产生一个唯一的数字值，该值可用于验证表内容的一致性。即它可以用来检查两个数据库之间同一张表是否一致，或者验证表随时间推移是否有变更。

## 语法

```sql
CHECKSUM TABLE tbl_name [, tbl_name] ... [QUICK | EXTENDED]
```
