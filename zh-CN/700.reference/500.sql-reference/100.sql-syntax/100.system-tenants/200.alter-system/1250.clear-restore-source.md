# CLEAR RESTORE SOURCE

## 描述

该语句用于撤销 `ADD RESTORE SOURCE` 语句加载的路径。

## 权限要求

执行该语句需要当前用户拥有 `ALTER SYSTEM` 权限。

## 语法

```sql
ALTER SYSTEM CLEAR RESTORE SOURCE;
```
