# DROP OUTLINE

## 描述

该语句用于删除 OceanBase 数据库中的 Outline。

## 语法

```sql
DROP OUTLINE outline_name;  
```

## 参数解释

|      参数      |         描述         |
|--------------|--------------------|
| outline_name | 指定要删除的 Outline 名称。 |

## 示例

删除 Outline `outline1`。

```sql
obclient> DROP OUTLINE outline1;      
Query OK, 0 rows affected
```
