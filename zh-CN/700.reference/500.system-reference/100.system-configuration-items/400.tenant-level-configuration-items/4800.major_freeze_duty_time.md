# major_freeze_duty_time 

## 功能描述

`major_freeze_duty_time` 用于设置每日定时冻结和合并的触发时刻。

## 属性说明

|      **属性**      |      **描述**      |
|------------------|------------------|
| 参数类型             | 时刻               |
| 默认值              | 02:00            |
| 取值范围             | \[00:00, 23:59\] 或 disable |
| 是否重启 OBServer 节点生效 | 否                |

## 使用说明

`major_freeze_duty_time` 的值为 `disable` 时，表示关闭每日合并。
