# user_tab_col_stat_cache_priority

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>由于调整了统计信息的维护方法，对应功能不再使用，该配置项从 V4.0.0 版本开始删除。</p>
</main>

## 功能描述

`user_tab_col_stat_cache_priority` 用于设置统计数据缓存在缓存系统中的优先级。

## 属性说明

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[1, +∞) |
| 是否重启 OBServer 节点生效 | 否        |
