USER_ALL_TABLES
====================================

功能
-----------

查看当前用户可访问的对象表和关系表，除了 OWNER 字段之外，其他字段和 ALL_ALL_TABLES 相同。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>
