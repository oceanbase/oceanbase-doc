# GV$OB_SQL_AUDIT

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示所有 OBServer 节点上每一次 SQL 请求的来源、执行状态等统计信息。
