# 查看 Locality 变更记录

本文介绍如何通过视图 `DBA_OB_TENANT_JOBS` 查看 Locality 的变更记录。

有关视图 `DBA_OB_TENANT_JOBS` 的详细介绍，参见 [DBA_OB_TENANT_JOBS](../../../../700.reference/700.system-views/400.system-view-of-mysql-mode/200.dictionary-view-of-mysql-mode/5700.oceanbase-dba_ob_tenant_jobs-of-mysql-mode.md) 。

## 操作步骤

1. 使用 root 用户登录到集群的 sys 租户。

   ```shell
   obclient -h172.30.xxx.xxx -P2883 -uroot@sys -pxxxx -A
   ```

2. 进入 `oceanbase` 数据库。

   ```shell
   obclient>use oceanbase;
   ```

3. 查看 Locality 变更任务。

   * 查看 Locality 变更任务的执行状态。
