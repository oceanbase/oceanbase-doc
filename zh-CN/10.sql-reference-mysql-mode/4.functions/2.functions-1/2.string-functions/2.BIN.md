BIN 
========================



声明 
-----------------------

```unknow
bin(N)
```



说明 
-----------------------

返回数字 `N` 的二进制形式。该函数等价于 `CONV(N,10,2)`。

如果 `N` 为 `NULL`，则返回 `NULL`。

示例 
-----------------------

```javascript
obclient> SELECT BIN(12);
+---------+
| BIN(12) |
+---------+
| 1100    |
+---------+
1 row in set (0.00 sec)
```
