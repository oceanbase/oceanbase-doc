connect_timeout 
====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/connect_timeout-1-2-3
`connect_timeout` 用于设置连接超时时间，单位为秒。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                                                                                                        |
| 默认值    | 10                                                                                                         |
| 取值范围   | \[2, 31536000\]                                                                                            |
| 生效范围   | Global   |
