server_cpu_quota_max 
=========================================

server_cpu_quota_max 用于设置系统可以使用的最大 CPU 配额。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值               | 1             |
| 取值范围             | \[1, 16\]     |
| 是否重启 OBServer 生效 | 是           |
