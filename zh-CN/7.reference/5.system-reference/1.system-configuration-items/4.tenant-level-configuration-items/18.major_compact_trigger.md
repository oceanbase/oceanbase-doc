major_compact_trigger 
==========================================

major_compact_trigger 用于设置多少次小合并触发一次全局合并。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 0            |
| 取值范围             | \[0, 65535\] |
| 是否重启 observer 进程生效 | 否            |



当 major_compact_trigger 配置项的取值为 0 时，表示关闭由转储次数触发的大合并，日常每日定时合并不受影响。
