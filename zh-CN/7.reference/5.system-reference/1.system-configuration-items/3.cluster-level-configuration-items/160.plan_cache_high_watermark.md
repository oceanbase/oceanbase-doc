plan_cache_high_watermark 
==============================================

plan_cache_high_watermark 用于设置执行计划缓存占用内存的阈值，如果超过该阈值时将触发自动淘汰。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>在当前版本中，该配置项已不再使用。</p>
  </main>


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 容量单位   |
| 默认值              | 2000M  |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |
