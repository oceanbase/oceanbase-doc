server_cpu_quota_min 
=========================================

server_cpu_quota_min 用于设置系统可以使用的最小 CPU 配额，系统会自动预留。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值               | 1              |
| 取值范围             | \[1, 16]       |
| 是否重启 OBServer 节点生效 | 是             |

"2.5", "[0,16]",
<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始默认值由 2.5 调整为 1，取值范围由 [0，16] 调整为 [1, 16]。</p>
</main>
