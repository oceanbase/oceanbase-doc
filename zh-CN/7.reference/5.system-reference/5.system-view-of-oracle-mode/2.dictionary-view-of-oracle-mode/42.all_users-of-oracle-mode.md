# ALL_USERS

## 功能

查看当前用户可见的所有数据库用户。

> **功能适用性**
>
> 该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。

## 相关视图

DBA_USERS

## 字段说明

| **字段名称** |    **类型**    | **是否可以为 NULL** | **描述**  |
|----------|--------------|----------------|---------|
| USERNAME | VARCHAR2(30) | NO             | 用户名     |
| USERID   | NUMBER(38)   | NO             | 用户 ID   |
| CREATED  | DATE         | NO             | 创建用户的时间 |
