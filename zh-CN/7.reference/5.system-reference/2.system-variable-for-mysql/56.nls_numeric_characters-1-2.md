nls_numeric_characters 
===========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/nls_numeric_characters-1-2

> **说明**
> 
> 该变量当前暂不生效。

| 参数类型    | varchar                                                                                                    |
| 默认值     | 空字符串                                                                                                       |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                                                                                                          |
