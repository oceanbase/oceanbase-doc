V$OB_SQL_AUDIT
===================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/v-sql_audit-3


功能
-----------

展示当前 OBServer 的每一次 SQL 请求的来源、执行状态等统计信息。该视图是按照租户拆分的，除了系统租户，其他租户不能跨租户查询。

---------------
