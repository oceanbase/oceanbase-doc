SYSTEM_USER 
================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/SYSTEM_USER


声明 
-----------------------

```sql
SYSTEM_USER()
```



说明 
-----------------------

返回当前的用户名与主机名，该函数是 `USER()` 的同义词。

示例 
-----------------------

```sql
obclient> SELECT SYSTEM_USER();
+---------------------+
| SYSTEM_USER()       |
+---------------------+
| root@xxx.xx.xxx.xxx |
+---------------------+
1 row in set 
```
