enable_major_freeze 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_major_freeze-1-2
enable_major_freeze 用于设置是否开启自动全局冻结的功能。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
