# DROP DIRECTORY

## 描述

该语句用来删除路径对象。

>**说明**
>
>删除路径对象必须具有 `DROP ANY DIRECTORY` 系统权限。

## 语法

```sql
DROP DIRECTORY directory_name;
```

## 参数解释

|       参数       |                            描述                            |
|----------------|----------------------------------------------------------|
| directory_name | 指定要删除的路径对象。 删除路径对象时，不会删除系统中相关联的共享文件系统路径。 |

## 示例

删除一个路径对象 `sql_file_dir`。

```sql
obclient> DROP DIRECTORY sql_file_dir;
Query OK, 1 row affected
```
