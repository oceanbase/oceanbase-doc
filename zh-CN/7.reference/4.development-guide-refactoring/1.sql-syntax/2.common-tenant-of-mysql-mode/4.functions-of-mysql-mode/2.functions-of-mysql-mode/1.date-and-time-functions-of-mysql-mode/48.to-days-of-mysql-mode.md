TO_DAYS 
============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/TO_DAYS


声明 
-----------------------

```javascript
TO_DAYS(date)
```



说明 
-----------------------

返回指定 `date` 值对应的天数。天数指距离 `0000-01-01` 的天数。

示例 
-----------------------

```javascript
obclient> SELECT TO_DAYS('2015-11-04'), TO_DAYS('20151104');
+-----------------------+---------------------+
| TO_DAYS('2015-11-04') | TO_DAYS('20151104') |
+-----------------------+---------------------+
|                736271 |              736271 |
+-----------------------+---------------------+
1 row in set 
```
