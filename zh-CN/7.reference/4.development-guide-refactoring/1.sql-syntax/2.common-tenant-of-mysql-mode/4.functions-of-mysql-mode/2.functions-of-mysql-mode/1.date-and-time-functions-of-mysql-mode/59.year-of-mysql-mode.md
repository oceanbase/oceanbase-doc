YEAR 
=========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/YEAR


声明 
-----------------------

```sql
YEAR(date)
```



说明 
-----------------------

返回 `date` 值的年份信息。

示例 
-----------------------

```sql
obclient> SELECT YEAR('1987-01-01');
+--------------------+
| YEAR('1987-01-01') |
+--------------------+
|               1987 |
+--------------------+
1 row in set 
```
