SIN 
========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/SIN


声明 
-----------------------

```sql
SIN(X)
```



说明 
-----------------------

返回给定弧度 `X ` 的正弦值。

示例 
-----------------------

```sql
obclient> SELECT SIN(PI());
+------------------------+
| SIN(PI())              |
+------------------------+
| 1.2246467991473532e-16 |
+------------------------+
1 row in set 
```
