default_password_lifetime 
==============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/default_password_lifetime-1-2-3

> **说明**
> 
> 该变量当前暂不生效。

| **属性**  |   **描述**    |
|---------|-------------|
| 参数类型    | uint        |
| 默认值     | 0           |
| 取值范围    | \[0,65535\] |
| 生效范围    | Global      |
| 是否参与序列化 | 是           |
