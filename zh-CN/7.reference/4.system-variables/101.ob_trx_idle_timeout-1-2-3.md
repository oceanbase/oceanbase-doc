ob_trx_idle_timeout 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_trx_idle_timeout-1-2-3
`ob_trx_idle_timeout` 用于设置事务空闲超时时间，即事务中两条语句之间的执行间隔超过该值时超时，单位为微秒。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 86400000000                                                                                                  |
| 取值范围    | \[100000000,+∞)                                                                                            |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                                                                                                          |
