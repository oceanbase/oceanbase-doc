query_cache_size
=====================================

#docslug#/oceanbase-database/oceanbase-database/V4.0.0/query_cache_size-1-2-3
query_cache_size 用于设置分配给存储历史查询结果的内存，即尚未使用的内存，单位为 Byte。

> **说明**
>
> 该变量当前暂不生效。

| **属性** |           **描述**            |
|--------|-----------------------------|
| 参数类型   | uint                        |
| 默认值    | 0                     |
| 取值范围   | \[0, 18446744073709551615\] |
| 生效范围   | GLOBAL                      |
