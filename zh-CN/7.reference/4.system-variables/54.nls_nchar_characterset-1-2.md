nls_nchar_characterset 
===========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/nls_nchar_characterset-1-2
`nls_nchar_characterset` 表示数据库默认字符集，用于 NCHAR、NVARCHAR2、NCLOB 等数据类型。

> **说明**
> 
> 该变量当前暂不生效。

|  **属性**  |  **描述**   |
|----------|-----------|
| 参数类型     | varchar   |
| 默认值      | AL16UTF16 |
| 取值范围     | N/A       |
| 生效范围     | Global    |
| 是否参与序列化  | 是         |
| 是否影响计划生成 | 是         |
