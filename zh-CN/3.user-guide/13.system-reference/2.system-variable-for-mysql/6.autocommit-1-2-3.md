autocommit 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/autocommit-1-2-3
autocommit 用于设置是否自动提交事务。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 1                                                                                                          |
| 取值范围    | <li> 1：自动提交   <li> 0：不自动提交    |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                                                                                                          |
