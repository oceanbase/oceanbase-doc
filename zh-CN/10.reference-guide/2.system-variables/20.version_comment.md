version_comment 
====================================

`version_comment` 用于查看 OBServer 的版本信息。


| **属性** |     **描述**      |
|--------|-----------------|
| 参数类型   | varchar         |
| 默认值    | OceanBase 3.1.0 |
| 取值范围   | N/A             |
| 生效范围   | Global          |
| 是否可修改  | 否               |
